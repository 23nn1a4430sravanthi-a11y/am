# Amazon Sales Dashboard

## Project Overview

This project is an interactive Power BI dashboard built using Amazon sales data.

## Key Insights

* Total Sales: 78.59M
* Maharashtra generated the highest sales.
* Shipped orders are the most common order status.
* Amazon fulfillment contributes the majority of orders.

## Dashboard Visualizations

* Sales by Category
* Sales by Size
* Sales by State
* Order Status Analysis
* Fulfillment Analysis
